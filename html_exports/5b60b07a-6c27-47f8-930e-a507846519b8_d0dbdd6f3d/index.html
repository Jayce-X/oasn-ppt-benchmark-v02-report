10.1 dictionary basics

Keys identify values.

A dictionary is a container object whose items are key-value pairs.

TypeThe Python type is dict.
KeysKeys cannot be duplicated inside the same dictionary.
ValuesValues are the data you look up by key.
Patternkey: value pairs are separated by commas.
example.py
days = {"Sunday": 1, "Monday": 2, "Tuesday": 3}

print(type(days))
print(days["Sunday"])
<class 'dict'> 1

Use curly braces for literals, or call dict() when converting pairs, keyword arguments, or another dictionary.

10.3 dictionary operations

Access values by key.

Square brackets are direct. get() is forgiving because it can return None or a default value when the key is missing.

dict[key]Use when the key should exist.
dict.get(key)Use when the key may be missing.
Defaultget(key, -1) can return a chosen fallback.
Case matters"Jaya" and "jaya" are different keys.
access.py
fruit = {"apple": 2, "banana": 3, "orange": 4}

print(fruit["banana"])
print(fruit.get("apple"))
print(fruit.get("pineapple", -1))
3 2 -1

10.3 dictionary operations

Look at keys, values, or whole items.

Dictionaries provide view objects through keys(), values(), and items(). Convert to list() when a list is needed.

views.py
codes = {"a": 97, "b": 98, "c": 99}

print(codes.keys())
print(list(codes.keys()))
print(codes.values())
print(codes.items())
dict_keys(['a', 'b', 'c']) ['a', 'b', 'c'] dict_values([97, 98, 99]) dict_items([('a', 97), ('b', 98), ('c', 99)])

10.4 conditionals

Ask membership questions.

Conditionals can check whether a dictionary contains a key, a value, or a full key-value item.

membership.py
fruit_count = {
    "apple": 2,
    "orange": 5,
    "pomegranate": 1
}

print("apple" in fruit_count.keys())
print(("orange", 5) in fruit_count.items())
print(fruit_count["pomegranate"] > 0)
True True True

Beginner note: in asks a yes/no question and returns a Boolean.

Membership checks prepare students for if-statements7 / 12

10.4 looping

Loop through a dictionary.

A for loop can visit keys, values, or key-value pairs.

Default loopfor key in dictionary:
Keysdictionary.keys()
Valuesdictionary.values()
Pairsdictionary.items()
looping.py
zip_codes = {
    "Berkeley": 94709,
    "Santa Cruz": 95064,
    "Mountain View": 94030
}

for city, code in zip_codes.items():
    print(city, code)
Berkeley 94709 Santa Cruz 95064 Mountain View 94030

Visual explanation

Each loop round gives the next item to the loop variables: first city, then code.

Use items() when both key and value matter8 / 12

10.5 nested dictionaries

Values can be dictionaries.

A nested dictionary stores another dictionary as a value. Access a nested value with the outer key, then the inner key.

nested.py
company_org_chart = {
    "Marketing": {"ID234": "Jane Smith"},
    "Sales": {
        "ID123": "Bob Johnson",
        "ID122": "David Lee"
    },
    "Engineering": {
        "ID303": "Radhika Potlapally",
        "ID321": "Maryam Samimi"
    }
}

print(company_org_chart["Sales"]["ID122"])
David Lee
Outer key first, inner key second9 / 12

10.5 dictionary comprehension

Create a dictionary from an iterable.

Dictionary comprehension transforms each element into a key-value pair.

comprehension.py
numbers = [1, 2, 3, 4, 5]

squares = {
    x: x**2
    for x in numbers
    if x % 2 == 0
}

print(squares)
{2: 4, 4: 16}

Syntax pattern: {key_expression: value_expression for element in iterable}

Comprehension is concise creation, not a replacement for every loop10 / 12

Runnable example 2 + practice

Count characters with a dictionary.

Loop through a string and use membership checks to update counts.

character_count.py
string_value = "This is a string"
char_count = {}

for character in string_value:
    if character in char_count:
        char_count[character] += 1
    else:
        char_count[character] = 1

print(char_count)
