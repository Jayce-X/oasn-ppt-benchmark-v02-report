Research problem and motivation

Generate controllable very large images without being trapped by a fixed training resolution.

02 / 08

Why the problem matters

  • Digital histopathology and remote sensing need precise, controllable synthesis beyond 1024 × 1024 pixels.
  • The paper frames this as useful for domains where large images encode global layout plus fine local structure.
  • The target scale reaches 4096 × 4096 in the experiments.

Finite diffusion limit

  • Pixel-space or latent-space conditional diffusion is designed around the resolution seen during training.
  • Scaling resolution makes compute grow substantially; the paper describes 4096 × 4096 training as infeasible for SDXL in its setup.
  • Examples named in the paper include SDXL and Matryoshka Diffusion at up to 1024 × 1024.

Patch-based trade-off

  • Patch methods can be more efficient by synthesizing local segments.
  • The downside stated by the paper: weak long-range spatial dependencies because far-away patches do not sufficiently affect each other.
  • ∞-Brush tries to keep the whole image as one function while sampling only sparse coordinates for training.

Key method idea

Represent an image as a function; condition the denoiser in function space.

03 / 08

What changes conceptually?

  • The dataset is written as pairs (uk, ek), where u is an image function and e is a control component.
  • Instead of training on every pixel, ∞-Brush samples coordinates x = {x(i)} and observes the function values there.
  • The same learned function-space model can be queried at arbitrary coordinate meshes during inference.

The operator that makes conditioning scalable

Move conditioning to sparse function samples with linear complexity in N.

05 / 08

Inputs to cross-attention

  • L = 3 condition streams: diffusion timestep t, condition embedding e, and coordinate embedding c.
  • Queries come from sparse point evaluations; keys and values come from each conditional embedding stream.
  • The paper's key implementation detail: multiply normalized keys and values pointwise before the query dot product.
Vanilla attention: O(N²d)
Cross-attention neural operator: O((N + ΣNl)d²)
Why it matters: at 4096 × 4096, N ≈ 16M if every coordinate is represented.
The paper states that the proposed operator is linear with respect to the number of points N.

Implementation details to remember

  • All ∞-Brush models are trained from scratch.
  • Each training iteration samples 256 × 256 = 65,536 pixels from the image.
  • Training uses Adam with learning rate 5e-5, EMA rate 0.995, DDIM with 50 inference steps.
  • Reported ∞-Brush training uses 4 NVIDIA A100 GPUs, batch size 20 per GPU.

Limitations stated by the paper

  • Images have better global structure consistency, but are not better than other methods in local details.
  • The paper hypothesizes that ∞-Brush has fewer parameters than Graikos et al. and is trained from scratch.
  • No infinite-dimensional pre-trained models are available for initialization in their setup.
  • Future work: local neural operators for fine details and transfer from finite-dimensional diffusion models.
